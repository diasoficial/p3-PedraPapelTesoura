# Pedra, Papel e Tesoura - Jogo Android

Este repositório contém um projeto de estudo focado na prática de desenvolvimento Android. O aplicativo implementa o clássico jogo "Pedra, Papel e Tesoura", utilizando `ConstraintLayout` para o layout e Java para a lógica do jogo.


## 📱 Sobre o Projeto

Este projeto foi criado para praticar e aprender sobre o desenvolvimento de aplicativos Android, com foco em:

- Manipulação de layouts utilizando XML.
- Uso de `ConstraintLayout` para criar interfaces responsivas.
- Implementação de lógica de jogo em Java.
- Manipulação de eventos e interatividade com o usuário.

O aplicativo permite que o usuário jogue contra o sistema, que faz escolhas aleatórias, e exibe o resultado da partida na tela.

## 🚀 Tecnologias Utilizadas

- **Linguagem:** Java
- **Plataforma:** Android
- **IDE:** Android Studio

## 📂 Estrutura do Projeto

- **MainActivity.java**: Contém a lógica principal do jogo, incluindo as escolhas aleatórias do sistema e a verificação de quem ganhou a rodada.
- **XML Layouts**: Define o design da interface do usuário utilizando `ConstraintLayout`.

## 🛠️ Funcionalidades

- Escolha entre Pedra, Papel ou Tesoura.
- Sistema faz uma escolha aleatória.
- Verifica e exibe o resultado da partida (vitória, derrota ou empate).
